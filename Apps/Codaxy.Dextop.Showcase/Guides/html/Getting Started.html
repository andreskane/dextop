Getting Started with Dextop

Before getting started with Dextop make sure you're familiar with Ext JS. Sencha provides a large number od demos and excellent documentation that you should not miss.

Source Code

Dextop source code is hosted at Google Code SVN repository. Please report any bug at the Google Code issue tracker.

Dox

Dextop API documentation is available here. To learn more about Dox please visit the Dox product page.

NuGets

Dextop offers a few NuGets packages. Creating new Dextop applications has never been easier. Check the 'NuGets' guide to learn how to quickly setup your first Dextop application.

Forum

For any questions regarding Dextop use the Community forum.
